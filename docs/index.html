AgentPlaybook

Run your coding agents with shared discipline.

AgentPlaybook bridges the gap between raw LLMs and clean engineering. It guides AI coding agents to prioritize local project rules, load targeted workflow cards dynamically, and always report verifiable evidence for their changes.

Routing

Smart, lightweight, selective routing.

Loading the entire playbook at once wastes tokens and degrades agent performance. Instead, our workflow router loads only the exact rules needed for the task—keeping context tight and execution razor-sharp.

01

Respect local rules

Local rules and configs are the ultimate source of truth for commands, architecture, and domain logic.

02

Find optimal path

Dynamically select paths tailored to features, bugfixes, code reviews, PRDs, releases, or documentation.

03

Load core cards only

Load only the specific common, platform, workflow, and product cards relevant to the current task step.

04

Report verification evidence

Every completed task requires verification logs or a detailed, honest explanation of blockers.
